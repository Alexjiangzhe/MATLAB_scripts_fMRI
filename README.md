# MATLAB Scripts for fMRI Analysis

This repository contains a collection of custom MATLAB scripts developed for functional Magnetic Resonance Imaging (fMRI) data processing and advanced analysis.

## Overview
The scripts in this repository cover various stages of fMRI analysis, including but not limited to:
* Data manipulation and parameter extraction
* Custom statistical modeling
* Multivariate pattern analysis (MVPA) and decoding tasks
